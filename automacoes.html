Faker Library

Gere arquivos CSV com nomes, CPFs, RGs e cartões de crédito estruturados. Ideal para popular bancos de dados de teste rapidamente.

Como funciona

Esta automação utiliza a biblioteca Faker no backend Python para forjar dados perfeitamente formatados. Nenhum dado real é exposto, garantindo adequação à LGPD nos seus testes.
